NAME

Webservice::OVH::Order::Hosting

SYNOPSIS

use Webservice::OVH;

my $ovh = Webservice::OVH->new_from_json("credentials.json");

my $info = $ovh->hosting->web->free_email_info;

DESCRIPTION

Only Helper Object to Web Api Sub-Object.

METHODS

_new

Internal Method to create the Hosting object. This method is not ment to be called directly.

  • Parameter: $api_wrapper - ovh api wrapper object, $module - root object, $type - intern type

  • Return: Webservice::OVH::Me::Task

  • Synopsis: Webservice::OVH::Me::Task->_new($ovh_api_wrapper, $type, $module);

web

Gives acces to the /order/hosting/web methods of the ovh api

  • Return: Webservice::OVH::Order::Hosting::Web

  • Synopsis: $ovh->order->hosting->web
