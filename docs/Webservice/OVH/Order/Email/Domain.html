NAME

Webservice::OVH::Order::Email::Domain

SYNOPSIS

use Webservice::OVH;

my $ovh = Webservice::OVH->new_from_json("credentials.json");

my $available_services = $ovh->order->email->domain->available_services;

DESCRIPTION

Provides the possibility to order MX packaged. The api methods are deprecated, but no alternative is give at the moment.

METHODS

_new

Internal Method to create the Domain object. This method is not ment to be called directly.

  • Parameter: $api_wrapper - ovh api wrapper object, $module - root object

  • Return: Webservice::OVH::Order::Email::Domain

  • Synopsis: Webservice::OVH::Order::Email::Domain->_new($ovh_api_wrapper, $module);

_new

Returns an array of available services.

  • Return: ARRAY

  • Synopsis: Webservice::OVH::Order::Email::Domain->_new($ovh_api_wrapper, $module);

allowed_durations

Returns an array of allowed durations. DEPRECATED

  • Parameter: $domain - target domain for MX package, $offer - MX offer

  • Return: ARRAY

  • Synopsis: $ovh->order->email->domain->allowed_durations('mydomain.de', '100');

allowed_durations

Returns information for a desired MX package. DEPRECATED

  • Parameter: $domain - target domain for MX package, $offer - MX offer, $duration - allowed duration

  • Return: ARRAY

  • Synopsis: $ovh->order->email->domain->info('mydomain.de', '100', $allowed_durations->[0]);

new

Generates an order for the desired MX package. DEPRECATED

  • Parameter: $domain - target domain for MX package, $offer - MX offer, $duration - allowed duration

  • Return: ARRAY

  • Synopsis: $ovh->order->email->domain->new('mydomain.de', '100', $allowed_durations->[0]);
